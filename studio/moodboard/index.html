Proyectos & Canvas
i
Crear proyectos
Desde la pantalla de inicio, pulsa "New Project". Cada proyecto puede tener varios canvas independientes.
i
Etiquetas (Tags)
Al crear o editar un proyecto puedes añadir etiquetas para filtrarlos en la pantalla de inicio.
i
Papelera
Los proyectos eliminados van a la pestaña "Trash". Desde ahí puedes restaurarlos o borrarlos definitivamente.
i
Guardado automático
Todo se guarda en tu navegador automáticamente cada 10 segundos. Exporta con el botón Export para hacer backup en un archivo .mb.
Herramientas de diseño
i
Añadir imágenes
Pulsa "Image" en la barra, arrastra desde tu ordenador, pega una URL o usa Ctrl+V con una imagen copiada.
Ctrl+V · Drag & Drop
i
Texto
Pulsa "Text" o la tecla T. Doble clic para editar. En el panel de propiedades puedes cambiar fuente, tamaño, estilo y color.
T · Doble clic para editar
i
Notas (Sticky Notes)
Pulsa "Note" o la tecla N para crear una nota rápida. Úsalas para ideas, observaciones o dirección creativa.
N · Doble clic para editar
i
Dibujar
Selecciona la herramienta "Draw" o pulsa D. Ajusta el tamaño y el color del trazo. Compatible con tableta y lápiz.
D · Ajustes en panel
i
Marco / Polaroid
Selecciona una imagen y pulsa "Frame". Elige entre marco estándar o estilo Polaroid con caption editable.
i
Recortar imagen
Selecciona una imagen, abre el panel de propiedades y pulsa "Crop". Ajusta el área azul y confirma con Apply.
i
Blanco y negro
Selecciona una imagen y aplica B&W desde el panel de propiedades para convertirla al instante.
i
Extraer colores
Selecciona una imagen y usa "Extract Colors". Aparecen 5 colores dominantes para aplicar o reutilizar en el canvas.
i
Comparador visual
Carga 2 a 4 imágenes para comparar paleta, tono, contraste y dirección visual. Úsalo para sacar una lectura común antes de montar el tablero final.
Capas & Organización
i
Panel de capas
Cada elemento va a la capa activa. Crea nuevas capas con el botón + y arrastra el handle para reordenarlas.
i
Renombrar capas
Doble clic sobre el nombre de una capa para cambiarlo por algo más descriptivo.
i
Bloquear capa
Activa el candado para impedir mover o editar los objetos de esa capa mientras trabajas.
i
Capas anidadas
Las capas padre organizan objetos y capas hijas debajo de ellas. Selecciona una capa para ver sus elementos y usa el orden visual para estructurar el board sin pensar en grupos clásicos.
Atajos de teclado
i
Navegación
V — Seleccionar · D — Dibujar
Espacio (mantener) — Mover canvas
T — Añadir texto · N — Añadir nota
Scroll — Zoom in/out
i
Edición
Ctrl+C — Copiar
Ctrl+V — Pegar (incluso entre canvas)
Ctrl+D — Duplicar en sitio
Ctrl+Alt+D — Duplicar en nueva capa
Supr/Backspace — Eliminar
i
Historial
Ctrl+Z — Deshacer
Ctrl+Y — Rehacer
Escape — Salir de modo actual
i
Modo Zen
El botón Zen de la topbar oculta la interfaz y deja el board en modo presentación limpia. Cambia de canvas desde la barra inferior. No muestra reglas ni guías y no permite editar.
Consejos pro
i
Clic derecho
Clic derecho sobre cualquier objeto para acceder a: Copiar, Pegar, Duplicar, Mover a nueva capa, Crop, B&W y más.
i
Panel de propiedades
Arrastra el panel desde su cabecera para moverlo. Pulsa el icono de fijar para anclarlo como sidebar derecho fijo.
i
Reglas y guías
Activa el botón de reglas para verlas en el canvas. Arrastra desde la regla superior para crear una guía vertical o desde la lateral para crear una horizontal. Las guías manuales son azules y las automáticas al mover objetos son rojas.
i
Export / Import .mb
Exporta tu workspace completo en formato .mb para hacer backup o transferirlo. Solo se puede abrir en MoodBoard Pro.
